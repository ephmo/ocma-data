OCMA-Data API

OCMA-Data is a structured Orthodox Calendar dataset providing JSON files for each liturgical day from 1924 to 2099, supporting both the Old Calendar (Julian) and New Calendar (Revised Julian).

It includes Feasts, Saints, Fasting rules, Lectionary information, Paschalion calculations, and Moon Phases. Data is fully multilingual with separate JSON files for English, Greek, Romanian and Russian.

API Examples

Below are example endpoints for accessing the dataset.

Daily Calendar (by Year)

Example:

  • Old Calendar 1924: /data/old/1924.json
  • New Calendar 2099: /data/new/2099.json

Multilingual Files

Each calendar day supports multiple languages.

  • English: /i18n/en.json
  • Greek: /i18n/el.json
  • Romanian: /i18n/ro.json
  • Russian: /i18n/ru.json

Paschalion

  • Old Calendar Paschalion: /data/paschalion/old.json
  • New Calendar Paschalion: /data/paschalion/new.json
